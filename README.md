# steiner_tree
## Сборка
~~~bash
git clone git@github.com:Mikhail038/steiner_tree.git
cd steiner_tree
make
~~~
## Использование
Базовый алгоритм:
~~~bash
./bin dat/{filename}.json 
~~~

Параллельный алгоритм:
~~~bash
./bin -m dat/{filename}.json 
~~~

Визуализация:
~~~bash
python3 viz.py dat/{filename}.json 
~~~

Общий тест:
~~~bash
./test_script.sh
~~~

Дополнительные тесты можно генерировать с помощью **test_gen.py**.

Общий тест в файлах **log.log** и **log-m.log** сохраняет информацию о времени выполнения и длине дерева.

## Модификация алгоритма
Я попробовал несколько вариантов, но самым простым и эффективным оказалась модфикация распараллеливания. При 8 потоках (по умолчанию) и 40 терминалах у меня получился выйгрыш в ~3 раза.
